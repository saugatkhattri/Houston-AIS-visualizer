# Houston AIS Visualizer

A simple demo maritime intelligence dashboard built using Python, Flask, Mapbox GL JS, and AIS vessel data.

This project demonstrates interactive ship tracking, playback simulation, route visualization, and maritime traffic analytics in the Houston Ship Channel area.

The repository is shared for learning, experimentation, and educational purposes. Anyone interested in GIS, Web Mapping, geospatial visualization, or maritime analytics can explore and learn from the implementation.

## Features

- AIS vessel playback
- Interactive Mapbox visualization
- Vessel route trails
- Traffic heatmap
- Playback controls
- Maritime analytics panel
- Real-time style dashboard interface

## Technologies

- Python
- Flask
- Mapbox GL JS
- HTML/CSS/JavaScript
- GeoJSON
- AIS Maritime Data

## Note

This is a demonstration and learning project and is not intended for operational maritime navigation or official vessel monitoring.
